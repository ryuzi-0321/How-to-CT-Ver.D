# Django セットアップスクリプト

Django の開発サーバーを起動するスクリプトです。

## 初回セットアップ（Windows）

### 1. 依存パッケージのインストール

```bash
pip install -r requirements.txt
```

### 2. データベース初期化

```bash
python manage.py migrate
```

### 3. 開発サーバーの起動

```bash
python manage.py runserver
```

### アクセス

ブラウザで以下のURLにアクセスしてください：

```
http://localhost:8000
```

## メニュー

- **ホーム** - システムトップページ
- **疾患検索** - 疾患データの検索・表示
- **お知らせ** - お知らせの表示
- **CTプロトコル** - CTプロトコルの参照

## 主な操作

### 疾患データの作成

1. ホームページの「新規疾患作成」をクリック
2. 疾患情報（疾患名、詳細）を入力
3. 撮影プロトコル、造影プロトコル、画像処理情報を入力（オプション）
4. 「作成」ボタンをクリック

### 疾患データの検索

1. 「疾患検索」ページに移動
2. 検索キーワードを入力して「検索」をクリック
3. 結果から疾患を選択して詳細を表示

### お知らせの投稿

1. ホームページの「新規お知らせ作成」をクリック
2. タイトルと内容を入力
3. 「作成」ボタンをクリック

## トラブルシューティング

### ポートが既に使用されている場合

```bash
python manage.py runserver 8001
```

別のポート（8001など）で起動できます。

### データベースをリセットしたい場合

```bash
# Windows
del medical_ct.db

# Linux/Mac
rm medical_ct.db

# その後、マイグレーション実行
python manage.py migrate
```

## 注意

- デバッグモード（DEBUG=True）のため、本番環境では使用しないでください
- 本番環境ではsettings.pyのSECRET_KEYを変更してください
- データベースファイル（medical_ct.db）は自動的に作成されます
